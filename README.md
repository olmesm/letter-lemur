**Note:** WIP

![Lemurs on loop](https://i.giphy.com/1ivxvG3WzSsoak7QNE.gif)

A React Typescript design libary inspired by the _awesome_ [Mailchimp Design System].

## Features

- React
- Typescript
- Storybook

## To Run

Please use [nvm].

```sh
# Match Node version
nvm use

# Install Dependencies
npm install

# Start Development Server
npm start

# Start Storybook
npm run storybook
```
